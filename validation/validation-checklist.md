# Baby Planner Validation Checklist

- [ ] Due date and LMP calculation works.
- [ ] Pregnancy progress updates correctly.
- [ ] Appointment data flows into the agent.
- [ ] Checklist progress flows into the agent.
- [ ] Agent asks follow-up questions when details are missing.
- [ ] Agent gives a prioritized plan.
- [ ] Agent produces doctor appointment questions.
- [ ] Agent checks readiness using the rubric.
- [ ] Streaming returns at least one tool event and one model text delta.
- [ ] UI displays progressive updates.
- [ ] Mobile layout works.
- [ ] Dark mode works.
- [ ] OPENAI_API_KEY is loaded only on the server.
- [ ] The app does not use deprecated Assistants API.
